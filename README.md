## 개요...
Factorize는 사실 Factor + -ize의 합성어로, (소)인수분해라는 뜻이 있습니다...만
여기서는
> Factory + ize = Factorize

로 **공장화하다**라는 의미로 썼습니다. 즉, 이 플러그인은 마인크래프트를 조금 더
효율적이고 조금 더 체계적으로 플레이할 수 있도록 도와주는 플러그인입니다.
